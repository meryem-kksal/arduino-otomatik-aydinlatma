# arduino-otomatik-aydinlatma
Işık sensörünün (LDR) ortamdaki ışık miktarına göre değişen analog değerini okudum. Oda karardığında eşik değerinin altına düşen voltajı yakalayıp trafik lambasını kademeli yaktım, ortam aydınlanınca söndürdüm.

Kullanılanlar: Arduino UNO, LDR (Işık Sensörü), 10k ohm Direnç, 3 kademeli lamba.
